# Archive

Material here is **historical or experimental**. It is not the default path for the CMU PokerAI tournament submission.

| Directory | Contents |
|-----------|----------|
| `old/` | Older engine/bot snapshots. |
| `old-bots/` | Legacy bot sources kept for reference. |
| `other-bots/` | Alternate bots and shared `support/` modules used by some experiments and by `apps/poker-rl-trainer` (opponent pool). |
| `scratch/` | Loose notes and one-off scripts (e.g. planning notes, renamed `brian_lam_player.py`). |

The canonical tournament agent and engine live at the repository root: `submission/player.py`, `agents/`, `match.py`, `run.py`, etc.
